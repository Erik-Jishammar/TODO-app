TODO App

A simple TODO app built with Express as the backend and a static frontend.  
Todos are stored in a local JSON file and managed through a small REST API.

Features

- Get all todos (`GET /todos`)
- Create a new todo (`POST /todos`)
- Update a todo (`PUT /todos/:id`)
- Delete a todo (`DELETE /todos/:id`)

git clone: https://github.com/Erik-Jishammar/TODO-app.git

cd TODO-app
npm install
npm start

Open browser at http://localhost:3000
